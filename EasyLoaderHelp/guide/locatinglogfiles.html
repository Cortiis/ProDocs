Locating Log Files

EasyLoader saves a log file called EasyLoader.log to your %AppData%\Local\Temp folder. This file contains the database connection information currently in use to upload data tables with. You can see the contents of this file from within EasyLoader by clicking Options and View Log.

If you experience difficulty uploading your data and you wish to review the upload settings that EasyLoader last used, then look at EasyLoader's recent upload log file. EasyLoader saves information about an upload session in a file called MapInfoEasyLoader.Recent.xml in the %AppData%\Roaming folder. This file lists the EasyLoader settings for the last upload, such as output folder location, connection settings, and data tables uploaded.

The AppData folder is in a path like C:\Users\<UserName>\AppData. If you cannot see the AppData folder, it is because it is hidden. To locate it, on the Microsoft Start menu in the Search programs and files field type %AppData%.
