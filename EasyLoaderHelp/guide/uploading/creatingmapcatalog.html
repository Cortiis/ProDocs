Unregister Tables from the Map Catalog

The Map Catalog stores the metadata used by MapInfo Pro and other MapInfo products to open tables with geometry data. If your database has a Map Catalog, you can delete entries in the catalog that are no longer valid.

Note: If the Map Catalog is present and there are no entries in it, the Catalog check box beside the table name in MapInfo EasyLoader does not enable.

To unregister entries from an existing Map Catalog:

  1. Select the data owner from the Owner Name drop-down list. The Tables Registered in Map Catalog list populates with columns of information for each registered table.
  2. Select the table or tables you want to unregister from the Map Catalog and then click the Unregister button.
  3. Click Close when you are done.

The Copy button copies the selected rows in a comma separated format to the clipboard, which you can then paste into other applications.
