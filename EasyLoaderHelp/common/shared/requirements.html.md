===================

<div class="body conbody">

-   Windows 10 64-bit
-   Windows 8.1 64-bit
-   Windows 8.1 32-bit
-   Windows 7 Ultimate 64-bit SP1
-   Windows 7 Ultimate 32-bit SP1
-   Windows 2012 Server R2 64-bit SP1
-   Windows 2008 Server R2 64-bit SP1
